# Competitor Intel

AI-powered competitor intelligence platform with ChangeDetection.io integration.

## Live Demo

**Production:** https://ts-competitor-intel.vercel.app

## Features

- **Site Monitoring** - Track competitor websites with CSS selector targeting
- **Visual Diff Viewer** - Side-by-side comparison of changes
- **Smart Alerts** - Email, Slack, and GHL webhook notifications
- **Change History** - Severity filtering and timeline view
- **GHL Integration** - Native Go High Level webhook support

## Tech Stack

- Next.js 15 with App Router
- TypeScript
- Tailwind CSS
- Radix UI Components
- Lucide React Icons

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Integration

This dashboard integrates with [ChangeDetection.io](https://changedetection.io/) for the actual monitoring. Configure your instance URL and API key in Settings.
